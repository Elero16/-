Угадай число (C++ версия)

Эта HTML-страница демонстрирует игру, написанную на C++.

Чтобы запустить настоящую C++ версию:

git clone https://github.com/ваш-логин/guess_number_game.git
cd guess_number_game
mkdir build && cd build
cmake .. && make
./game
